# Zikir Vakti - Specialized Spiritual Tracking App

This is a unique Android application that bridges traditional spiritual practices with complex astronomical calculations. 

### 🌌 The Unique Algorithm
Unlike standard tasbih apps, **Zikir Vakti** features a proprietary calculation engine that:
* Maps the **99 Names of Allah (Asma-ul Husna)** to specific **Planetary Hours** .
* Calculates precise time intervals based on astronomical data to determine the optimal "dhikr" periods for the user.
* Operates as the first digital tool to integrate this specific traditional calculation method into a mobile interface.

### 🛠️ Technical Implementation
* **Language:** Kotlin & XML
* **Algorithm:** Astronomical time calculation & specific data mapping for planetary alignments.
* **Platform:** Published on Google Play Store.
